📊 WHAT HAPPENS WHEN YOU ASK:

  1. You type question in browser
  2. JavaScript sends to backend
  3. Python processes the query
  4. TF-IDF matching searches training data
  5. Fuzzy matching searches knowledge base
  6. Best answer returned
  7. JavaScript displays response
  8. Timestamp added automatically
