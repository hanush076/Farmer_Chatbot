📊 API ENDPOINTS

  1. POST /api/chat
     └─ Send message, get response
        Request:  {"message": "Question here"}
        Response: {"success": true, "response": "Answer here"}

  2. GET /api/categories
     └─ Get available categories
        Response: {"categories": [...]}

  3. GET /api/help
     └─ Get help information
        Response: {"help": "Help text..."}

  4. GET /api/health
     └─ Check server status
        Response: {"status": "online", "chatbot": "ready"}

🐛 TROUBLESHOOTING

  Problem: Page not loading
  Solution: Refresh browser (Ctrl+R) or check if server is running

  Problem: No response
  Solution: Check browser console (F12) or server logs

  Problem: Port in use
  Solution: Change port in app.py or kill existing process

  Problem: SSL warnings
  Solution: Normal on macOS, doesn't affect functionality
